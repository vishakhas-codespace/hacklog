
# 🔐 HackLog — Your Cybersecurity Learning Journal

A terminal-based cybersecurity learning journal built with Python.
Built by a hacker in the making.

## What it does

- 📝 Log what you learn every day with timestamps
- 📖 View your entire learning journey
- 🎯 Take quick cybersecurity quizzes
- ⚡ Get daily cybersecurity challenges
- 📡 Shows a random AI × Cybersecurity fact on startup

## How to run

```bash
python hacklog.py
```

## Requirements

- Python 3.10+
- No external libraries needed

## Why I built this

I'm a BSc IT student learning AI × Cybersecurity from scratch.
HackLog is how I document every step of the journey.

## Tech used

- Python
- File I/O
- Dictionaries, Lists, Functions, Strings
